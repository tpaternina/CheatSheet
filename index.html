HTML Elements

Tag Name Description
<article> Article Creates a standalone composition in the web page, independent from other elements, to be re-used and re-distributed by other applications or websites.
<section> Section Creates a standalone section inside a page, when no other more semantic element can be used
<div> Division Generic container for any kind of content. Use whenever section or other more semantic element cannot be used.

CSS Properties

Property Description Values
margin The space surrounding the outside of an element. top right bottom left. Values can be in pixels, percentages, or em.
padding The space between the content and the borders of an element. top right bottom left. Values can be in pixels, percentages, or em.
border The "lines" wrapping an element. thickness style color
font-family The font family name(s), or the generic name of family types, to display text content. name1[, name2, name3 ...]. Can be specific names (e.g. Arial, Georgia, Times), or generic names (e.g. sans-serif, serif).
